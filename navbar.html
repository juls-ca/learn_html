Learning Nav Bar

One use of lists is to create navigation bars. Lists provide the structure of the nav bar.
We can wrap the whole nav bar using the <nav> element which will mark up elements that are part of the site navigation.
It is not used all the time, but mainly for major blocks of navigation.
